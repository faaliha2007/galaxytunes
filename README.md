# 🌌 GalaxyTunes

 **Live Application:**

## 🚀 Live Concept

**GalaxyTunes** allows users to:

- 🎶 View all available tracks  
- ⭐ Add / Remove tracks from favourites  
- 🌠 See liked tracks instantly across pages  
- 🏆 Explore a leaderboard view  
- ⚡ Experience smooth animations and UI transitions  

All updates happen instantly without page reload using **React Router**.

---

## 🛠️ Tech Stack

- ⚛️ **React (Functional Components)**
- 🔁 **React Router DOM**
- 🌐 **useContext (Global State Management)**
- 🎨 **Tailwind CSS**
- 💫 **CSS Animations**

---

## 🧠 Core Concepts Used

### 1️⃣ Global State Management

- Created `TrackContext` using `createContext()`
- Used `useContext()` to access favourites across pages
- Prevented duplicate favourites

---

### 2️⃣ Routing

- `/` → All Tracks  
- `/favourites` → Liked Tracks  
- `/leaderboard` → Track Ranking  

Navigation happens without reloading the page.

---

### 3️⃣ Dynamic Rendering

- Track data stored in an array of objects  
- Rendered dynamically using `.map()`  
- Conditional star icon rendering ⭐
